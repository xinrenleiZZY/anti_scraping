数据流

scraper_config.json
       ↓
auto_amazon_scraper.py → amazon_data/*.json
       ↓
dataprocess.py         → processed_data/*_processed.json
       ↓
pipeline._save_to_database() → PostgreSQL
API 端点
方法	路径	说明
GET	/	健康检查
GET	/api/results	查询爬取结果，支持 ?keyword=
GET	/api/tasks	查询任务列表
POST	/api/scrape	触发爬取，支持 ?keyword=&pages=
手动运行流程

# 1. 改关键词
# 编辑 backend/app/scraper/scraper_config.json

# 2. 爬取
cd backend/app/scraper
python auto_amazon_scraper.py

# 3. 预处理
python dataprocess.py

# 4. 入库（通过 API）
curl -X POST http://localhost:8000/api/scrape
