# 文件解析

## 根目录

| 文件 | 作用 |
|------|------|
| `.env` | 实际运行环境变量（数据库密码、端口等），不提交git |
| `.env.example` | 环境变量模板，供新成员参考配置 |
| `.env.worker.example` | Worker节点环境变量模板 |
| `config.json` | 爬虫关键词配置（根目录备份版本） |
| `docker-compose.yml` | 主服务编排：backend + frontend + postgres |
| `docker-compose.worker.yml` | Worker节点独立部署编排文件 |
| `init.sql` | PostgreSQL初始化SQL（建表、初始数据） |
| `init-renyuan.sql` | 人员相关初始化SQL |
| `README.md` | 项目说明文档 |
| `文件解析.md` | 本文件，所有文件路径与作用说明 |
| `amazon_scraper_old.log` | 旧版爬虫运行日志（历史存档） |

## backend/

| 文件 | 作用 |
|------|------|
| `Dockerfile` | 后端容器构建文件 |
| `requirements.txt` | 主服务Python依赖列表 |
| `requirements_worker.txt` | Worker节点Python依赖列表 |
| `worker_daemon.py` | Worker守护进程，接收并执行分布式爬取任务 |
| `123.py` | 临时测试脚本（可清理） |
| `amazon_scraper.log` | 当前爬虫运行日志 |
| `amazon_scraper_old.log` | 旧版爬虫日志（历史存档） |
| `debug_china_page.html` | 调试用：抓取到的亚马逊页面HTML快照 |

### backend/migrations/

| 文件 | 作用 |
|------|------|
| `add_keyword_attributes.py` | 数据库迁移脚本：为关键词表添加属性字段 |

### backend/app/

| 文件 | 作用 |
|------|------|
| `__init__.py` | 包初始化 |
| `main.py` | FastAPI应用入口，注册所有路由，启动服务 |
| `config.py` | 配置管理，读取.env环境变量，统一配置对象 |
| `database.py` | PostgreSQL连接池与会话管理（SQLAlchemy） |
| `models.py` | SQLAlchemy ORM数据表定义（产品、关键词、任务等） |
| `schemas.py` | Pydantic模型，定义API请求/响应数据格式 |
| `crud.py` | 数据库增删改查操作函数 |

### backend/app/api/

| 文件 | 作用 |
|------|------|
| `__init__.py` | 包初始化 |
| `keywords.py` | 关键词管理API（增删改查，数据库版） |
| `keywords_json版本.py` | 关键词管理API旧版（操作JSON文件，已弃用） |
| `scraping.py` | 爬取任务API（启动爬取、查询任务状态） |
| `data.py` | 数据查询API（产品列表、分析报告、仪表盘统计） |
| `data_json版本.py` | 数据查询API旧版（读取JSON文件，已弃用） |
| `distributed.py` | 分布式任务API（下发任务到Worker节点） |
| `logs.py` | 日志查询API（返回爬虫运行日志） |
| `users.py` | 用户管理API（登录、权限等） |

### backend/app/distributed/

| 文件 | 作用 |
|------|------|
| `task_manager.py` | 分布式任务调度核心：任务分发、状态追踪、Worker管理 |

### backend/app/scraper/

| 文件 | 作用 |
|------|------|
| `scraper_config.json` | 爬虫字段配置（需求字段定义，部署到数据库用） |
| `keywords_config.json` | 关键词列表配置文件 |
| `anti_scraping_config.py` | 反爬虫策略配置（延迟、重试、IP轮换参数） |
| `headers_manager.py` | HTTP请求头管理，随机UA、Cookie等 |
| `auto_amazon_scraper.py` | 原始独立爬虫脚本（项目集成前的版本） |
| `scraper.py` | 项目核心爬虫（curl_cffi + BeautifulSoup，绕过反爬） |
| `dataprocess.py` | 原始数据预处理脚本（清洗后存入源数据库） |
| `data_processor.py` | 结构化数据处理器（字段映射、清洗、入库） |
| `pipeline.py` | 完整爬取流程编排（爬取→处理→存储） |
| `check_version_curl_cffi.py` | 检测curl_cffi版本兼容性的工具脚本 |
| `amazon_scraper.log` | 爬虫模块运行日志 |
| `功能介绍.md` | 爬虫模块功能说明文档 |
| `字段映射.md` | 爬取字段与数据库字段的映射关系说明 |
| `随机USIP.md` | 随机US IP策略说明文档 |
| `amazon_data/` | 爬取原始数据目录（csv/json/report，按关键词+时间戳命名） |
| `processed_data/` | 处理后的结构化数据目录（_processed.json） |
