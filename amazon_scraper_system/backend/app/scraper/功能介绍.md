## 📋 爬取控制功能概览

`scrape.html` + `scrape.js` 提供了两种爬取方式：

| 功能 | 说明 |
|------|------|
| **手动爬取** | 用户选择关键词和页数，立即执行爬取 |
| **每日任务** | 每天早上9点自动爬取所有关键词 |
| **每周任务** | 每周一早上9点自动爬取所有关键词（更多页数） |
| **实时状态** | 显示正在运行的任务 |
| **实时日志** | 显示爬取过程的日志输出 |

### 代码实现

#### 前端 `scrape.js` - 手动爬取

```javascript
window.startScrape = async function() {
    const keyword = document.getElementById('scrapeKeyword').value;  // 获取选中的关键词
    const pages = document.getElementById('scrapePages').value;      // 获取页数
    
    let url = '/scrape';
    const params = new URLSearchParams();
    if (keyword) params.append('keyword', keyword);  // 不选则爬取所有
    if (pages) params.append('pages', pages);        // 不选则自动获取所有页
    if (params.toString()) url += '?' + params.toString();
    
    // 显示等待状态
    statusDiv.innerHTML = '<div class="alert alert-info">任务已提交，正在执行...</div>';
    
    // 调用后端 API
    const result = await apiFetch(url, { method: 'POST' });
    statusDiv.innerHTML = `<div class="alert alert-success">${result.message}</div>`;
};
```

#### 后端 `scraping.py` - 接收请求

```python
@router.post("/scrape")
async def start_scraping(
    keyword: Optional[str] = Query(None),
    pages: Optional[int] = Query(None)
):
    """启动爬取任务"""
    async def run_in_thread():
        from app.scraper.pipeline import run_now
        result = await asyncio.to_thread(run_now, keyword, pages)
        return result

    # 创建异步任务，不阻塞主线程
    task = asyncio.create_task(run_in_thread())
    return {
        "message": "任务已启动",
        "keyword": keyword or "所有关键词",
        "pages": pages or "自动"
    }
```

#### 后端 `pipeline.py` - 核心爬取流程

```python
def run_now(keyword: str = None, pages: int = None) -> Dict:
    """
    手动触发爬取
    
    - keyword: 关键词，不传则爬取配置文件中的所有关键词
    - pages: 页数，不传则自动获取总页数
    """
    pipeline = ScrapingPipeline()
    try:
        if keyword:
            return pipeline.run_full_pipeline(keyword, pages)  # 单关键词
        else:
            return pipeline.run_batch(pages)                   # 批量爬取
    finally:
        pipeline.close()
```

### 使用示例

| 选择 | 行为 |
|------|------|
| 选择"所有关键词" | 爬取 `scraper_config.json` 中所有关键词 |
| 选择具体关键词 | 只爬取该关键词 |
| 不填页数 | 自动检测搜索结果总页数并全部爬取 |
| 填写页数（如3） | 只爬取前3页 |

## 2️⃣ 定时任务

### 工作流程

```
系统时间到达 9:00                   后端处理
       │                                 │
       ▼                                 │
  ┌─────────────┐                        │
  │  APScheduler │                       │
  │  检测到时间   │                       │
  └──────────────┘                       │
       │                                 │
       ▼                                 │
  ┌─────────────┐     POST /scrape/daily │
  │ 触发每日任务  │ ─────────────────────>│
  └──────────────┘                       │
                                         ▼
                                  ┌─────────────┐
                                  │  run_daily() │
                                  │  批量爬取    │
                                  └──────────────┘
```

### 两种定时任务

| 任务类型 | 触发时间 | 行为 | 代码 |
|----------|----------|------|------|
| 每日任务 | 每天 9:00 | 爬取所有关键词（默认页数） | `run_daily()` |
| 每周任务 | 每周一 9:00 | 爬取所有关键词（自动获取所有页） | `run_weekly()` |

### 后端实现 `pipeline.py`

```python
def run_daily():
    """每日爬取任务"""
    print(f"=== 每日爬取任务开始: {datetime.now()} ===")
    pipeline = ScrapingPipeline()
    try:
        results = pipeline.run_batch()  # 批量爬取所有关键词
        success_count = len([r for r in results if r['status'] == 'success'])
        print(f"=== 每日任务完成: 成功 {success_count}/{len(results)} ===")
        return results
    finally:
        pipeline.close()

def run_weekly():
    """每周爬取任务"""
    print(f"=== 每周爬取任务开始: {datetime.now()} ===")
    pipeline = ScrapingPipeline()
    try:
        results = pipeline.run_batch(pages=None)  # 自动获取所有页
        success_count = len([r for r in results if r['status'] == 'success'])
        print(f"=== 每周任务完成: 成功 {success_count}/{len(results)} ===")
        return results
    finally:
        pipeline.close()
```

## 总结

| 功能 | 触发方式 | 后端处理 |
|------|----------|----------|
| 手动爬取（单关键词） | 选择关键词后点击按钮 | `run_full_pipeline(keyword, pages)` |
| 手动爬取（所有关键词） | 不选关键词直接点击 | `run_batch(pages)` |
| 每日任务 | 定时器 9:00 或手动触发 | `run_daily()` → `run_batch()` |
| 每周任务 | 定时器 周一 9:00 或手动触发 | `run_weekly()` → `run_batch(pages=None)` |

**所有爬取任务都是异步执行的**，不会阻塞前端页面，用户可以通过实时日志和任务状态监控爬取进度！
